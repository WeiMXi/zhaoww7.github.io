一、天河二号简介

研发历程：2013年6月，经过280多人长达2年多的潜心研发，天河二号正式研制成功，随即登上TOP500榜单首位，一直保持到2016年初。中国超算的前世今生

整体结构：天河二号共有170个机柜，占地720平方米，包含计算阵列、服务阵列、存储子系统、互联通信子系统、监控诊断子系统等，如图所示。

计算节点：计算阵列全系统包含125个计算机柜，每个计算机柜包含4个计算插框，每个计算插框包含16个计算刀片，每个计算刀片包含2个计 算节点，因此计算阵列一共包含16000个计算节点。每个计算节点包含：2个Intel Xeon E5-2692 v2@2.2GHz 12核心处理器、 3个Intel Xeon Phi 31S1P 57核心协处理计算卡、64GB主存、高速互联接口、2个以太网接口。由于每个Xeon Phi处理器带有8G内存， 所以每个计算节点有88G内存，整体内存约为1.34PB。一个Xeon E5处理器峰值浮点计算能力为0.2112 TFlops， 一个Xeon Phi协处理器峰值浮点计算能力为1.003 TFlops，单节点峰值性能为3.4314 Tflops，计算阵列理论峰值浮点计算能力为54902.4 TFlops， 也就是每秒钟5.49亿亿次。天河二号计算节点运行Red Hat Enterprise Linux Server release 7.3 (Maipo)操作系统， 采用基于SLURM(Simple Linux Utility for Resource Management，简称SLURM)研发的作业调度系统，支持MPI编程，可以使用gcc、icc编译。

服务领域：新材料新能源、生物医疗健康、天文地理物理、工业设计制造、大气海洋环境和智慧城市云计算等方面的软件，在国防建设、科技创新、国民经济发展等各个方面发挥了重要作用

重大成果：中国商用飞机设计有限公司北京研究中心利用约2.4万CPU核开展了大型民机全参数气动优化设计；华大基因互联网支撑与发展中心负责人说，天河二号具有强大的计算能力，以500人规模的全基因组信息关联性分析 宇宙中微子数值模拟；

二、天河二号账号申请

1.VPN

每个用户组（包含多个用户）拥有一个VPN账号，如果导师开通了主账户，学生开通子账户，那么向导师获取VPN账号即可。

2.天河二号账号

子账户需要自己申请。给天河二号相关人员写申请邮件，并且抄送给导师或相关老师。

发送邮件之后，一般几天后就能收到回复，如果申请通过的话，回复邮件包含：新账号用户名、登录秘钥、使用说明书等
